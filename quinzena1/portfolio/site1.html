Site - FKBERS

Este foi meu ultimo projeto, fiz para o mesmo cliente do site da Academia Cobra Bushidoryu. Para deixar o layout responsivo, utilizei Media Query. O projeto é bem simples, mas cumpre o seu objetivo. Pretendo fazer alguns ajustes futuramente mas apesar disso o cliente ficou contente com o resultado.

Tecnologias utilizadas

Utilizei HTML5 para construir as páginas do website.
Utilizei CSS3 para estilizar e aplicar animações da galeria de fotos as páginas do website.
Utilizei Javascript para aplicar animações da galeria de fotos e para o button do menu responsivo.
O Photoshop foi utilizado para recortar imagens e aplicar transparências.
O código foi desenvolvido no Visual Studio Code.
