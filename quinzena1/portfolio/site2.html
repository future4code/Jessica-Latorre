Tecnologias utilizadas

Utilizei HTML5 para construir as páginas do website.
Utilizei CSS3 para estilizar e aplicar animações da galeria de fotos as páginas do website.
O Photoshop foi utilizado para recortar imagens e aplicar transparências.
O código foi desenvolvido no Visual Studio Code.
Neste projeto utilizei o Bootstrap. Na época estava estudando por conta própria sobre o Bootstrap e resolvi testar isso.
